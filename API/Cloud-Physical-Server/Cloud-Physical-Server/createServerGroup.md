# createServerGroup


## Description
Create virtual server group

## Request Method
PUT

## Request Address
https://cps.jdcloud-api.com/v1/regions/{regionId}/serverGroups

|Name|Type|Required or Not|Default Value|Description|
|---|---|---|---|---|
|**regionId**|String|True| |Region ID, call APIs (queryCPSLBRegions) to get regions supported by Cloud Physical Server|

## Request Parameter
|Name|Type|Required or Not|Default Value|Description|
|---|---|---|---|---|
|**clientToken**|String|False| |Generated by the client to guarantee idempotence of request, and the length cannot exceed 36 characters;<br/><br>if multiple requests use the same clientToken, only the first request will be executed and the following requests will directly return the result of the first request<br/><br>|
|**serverGroupSpec**|ServerGroupSpec|True| |Virtual Server Group Configuration|

### ServerGroupSpec
|Name|Type|Required or Not|Default Value|Description|
|---|---|---|---|---|
|**loadBalancerId**|String|False| |Load Balancer ID|
|**name**|String|False| |Virtual Server Group Name|

## Return Parameter
|Name|Type|Description|
|---|---|---|
|**result**|Result| |
|**requestId**|String| |

### Result
|Name|Type|Description|
|---|---|---|
|**serverGroupId**|String|ID of Server Group|
