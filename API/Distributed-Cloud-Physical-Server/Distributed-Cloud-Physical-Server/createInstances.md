# createInstances


## Description
Create one or more Distributed Cloud Physical Servers with specified configuration<br/>
- Region and Availability Zone<br/>
  - The region and availability zone supported by the Distributed Cloud Physical Servers can be obtained by calling API (queryEdCPSRegions)<br/>
- Instance Type Family<br/>
  - The instance type family list can be obtained by calling API (describeDeviceTypes)<br/>
  - Instance type family which is off line or sold out is not available for use<br/>
- Operating System<br/>
  - The API (describeOS) may be called to obtain the operating system list supported by Distributed Cloud Physical Server<br/>
- Storage<br/>
  - Multiple RAIDs are available for the data disk and the RAID list supported by the server can be obtained by calling API (describeDeviceRaids)<br/>
- Network<br/>
  - Network type supports vpc currently<br/>
  - The ISP Line now supports China Unicom (un), China Telecom (ct) and China Mobile (cm)<br/>
  - Disabled Internet is supported. If the Internet is enabled, the bandwidth range is [1,200], in unit Mbps<br/>
- Others<br/>
  - Purchase Duration, which can be purchased by Monthly Package: month value range [1, 9] and year value range [1, 3]<br/>
  - Refer to the Public Parameter Specification for password setting<br/>


## Request Method
PUT

## Request Address
https://edcps.jdcloud-api.com/v1/regions/{regionId}/instances
