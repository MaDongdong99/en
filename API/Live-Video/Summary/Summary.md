Live Streaming API Overview
=================================

        Live-Video is based on JD Cloud's large-scale cloud computing infrastructure resources, providing customers with low-latency, high-availability and easy-to-access live video one-stop services,
    Helping customers quickly build their own live platforms and applications. After enabling the JD Cloud Live Acceleration Service, the customers can use the live Open API provided by JD Cloud to
    manage their own live broadcast business in a more convenient and multi-dimensional way.      
        The version number of the current live Open API is: 1.0.1
