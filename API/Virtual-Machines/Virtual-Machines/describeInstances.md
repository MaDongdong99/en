# describeInstances


## Description
Batch query detailed information of VMs.
This API supports paging query with 20 items per page by default.


## Request method
GET

## Request address
https://vm.jdcloud-api.com/v1/regions/{regionId}/instances

|Name|Type|Required or not|Default value|Description|
|---|---|---|---|---|
|**regionId**|String|True| |Region ID|

## Request parameter
|Name|Type|Required or not|Default value|Description|
|---|---|---|---|---|
|**pageNumber**|Integer|False|1|Page; 1 by default|
|**pageSize**|Integer|False|20|Paging Size; 20 by default; Value range[10, 100] |
|**filters**|Filter[]|False| |instanceId - Virtual Machines ID, accuracy matching, supporting multiple <br>privateIpAddress - Primary network interface Intranet primary IP address, fuzzy matching, support multiple <br>az - Availability zone, accuracy matching, supporting multiple<br>vpcId - Virtual Private Cloud ID, accuracy matching, support multiple<br>status - Virtual Machines Status, accuracy matching, support multiple <a href="http://docs.jdcloud.com/virtual-machines/api/vm_status">Refer to Virtual Machines Status</a><br>name - Virtual Machines Name, fuzzy matching, support single<br>imageId -Image ID, accuracy matching, support multiple <br>networkInterfaceId - Elastic Network Interface ID, accuracy matching, support multiple<br>subnetId - Subnet ID, accuracy matching, support multiple<br>instanceType - Instance Type, accuracy matching, support multiple<br>agId - Support single by using the availability group id<br>faultDomain - Fault domain, support multiple<br>|

### Filter
|Name|Type|Required or not|Default value|Description|
|---|---|---|---|---|
|**name**|String|True| |Name of Filter Requirements|
|**operator**|String|False| |Operator of filter requirements is eq by default|
|**values**|String[]|True| |Value of Filter Requirements|
