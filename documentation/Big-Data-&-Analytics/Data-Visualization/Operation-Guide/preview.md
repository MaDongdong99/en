#### 8. Preview/Release of Data Visualization

You can view the Data Visualization by clicking **Preview / Release** in the upper right corner of the console of the Data Visualization.

![](https://github.com/jdcloudcom/cn/blob/edit/image/Data-Visualization/media/a082f3f48a15c524e600aabe3a5979a5.png)



When you click **Release**, the release configuration page will pop up in the system. In this page, you can maintain the encrypted passwords.

![](https://github.com/jdcloudcom/cn/blob/edit/image/Data-Visualization/media/5e248bfd8c7d9b5fb6b164fe0361579e.png)



For Data Visualizations that have already been released, you can share them with URL. In the case of encryption sharing, the user needs to enter the password to view the Data Visualization.

![](https://github.com/jdcloudcom/cn/blob/edit/image/Data-Visualization/media/5d7be174b947d1d73e816ff2793b211f.png)



After the user enters the password, he/she can view the Data Visualization in full screen.

![](https://github.com/jdcloudcom/cn/blob/edit/image/Data-Visualization/media/4ffadbaf3dba6f9002dfa514fc98aae8.png)
