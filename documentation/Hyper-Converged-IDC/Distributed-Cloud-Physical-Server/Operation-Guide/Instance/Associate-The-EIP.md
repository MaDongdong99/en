## Associate Elastic IP

- Access [Instance Console](https://cps-edge-console.jdcloud.com/instance/list) or access [JD Cloud & AI Console](https://console.jdcloud.com/overview), click **AIDC - Distributed Cloud Physical Server** in the left navigation bar and log in the Instance List Page, then you can view the instance information.
, click **Distributed Cloud Physical Server** - **Instance List** page in turn in the left navigation bar, select a target instance, and click **Associate Elastic IP** in the operation to pop up the information;<br/>

- Select the network interface name and target private IP as needed, select an EIP that can be associated, and click **OK** to complete the association.<br/>

  Note: You can only disassociate the EIP associated by the primary network interface in the instance list.
