# View Bandwidth (Exit) List

Open the Console, select **Hyper-Converged IDC** ->**Edge Physical Computing Service ->**Resource Management**->**Bandwidth (Exit)** to view the list of bandwidth (exit) traffic, as shown in the following figure:</br>
Click **Edge Node** on the top. The nodes can be switched over to view the bandwidth (exit) traffic lists of different nodes.
![带宽列表查看连接](https://github.com/jdcloudcom/cn/blob/cn-edge-physical-computing-service/documentation/Hyper-Converged-IDC/Edge-Physical-Computing-Service/Image/EPCSBW001.png)


# View Bandwidth (Exit) Details
Click **Bandwidth (Exit) Name**, redirect to Bandwidth (Exit) Details, which displays two parts including Basic Information and Associated Public IP Information, as shown in the following figure:</br>
