# View Bandwidth (Exit) List

Open the Console, select Edge Physical Computing Service->Resource Management->Bandwidth (Outlet), view the bandwidth (outlet) list, click **Edge Node** on the top to switch nodes and view the bandwidth (outlet) list at different nodes. The list of bandwidth (outlet) includes: bandwidth (outlet) name, associated public IP, line type, etc.



# View Bandwidth (Exit) Details
Click **Bandwidth (Outlet) Name** to redirect to the details of bandwidth (outlet) displaying two parts, including basic information and associated public IP information.
