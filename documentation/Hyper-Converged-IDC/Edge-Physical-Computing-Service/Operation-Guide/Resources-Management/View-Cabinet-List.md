# View cabinet list

Open the Console, select **Hyper-Converged IDC** ->**Edge Physical Computing Service ->**Resource Management** ->**Cabinet** to view the cabinet list, as shown in the following figure:</br>
Click **Edge Node** on the top. The nodes can be switched over to view the cabinet lists of different nodes.
![机柜列表查看连接](https://github.com/jdcloudcom/cn/blob/cn-edge-physical-computing-service/documentation/Hyper-Converged-IDC/Edge-Physical-Computing-Service/Image/EPCS003.png)

# View Cabinet Visual View
Click **Cabinet View**, and view the visual display of cabinet, shown as the following figure:
