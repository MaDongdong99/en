# View list of Public IPs
Open the Console, select **Hyper-Converged IDC** ->**Edge Physical Computing Service ->**Resource Management** ->**Public IP** to view the Public IP lists, as shown in the following figure:</br>
Click **Edge Node** on the top. The nodes can be switched over to view the Public IP lists of different nodes.
