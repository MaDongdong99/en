# Application Scenarios

The applicable scenarios for the IoT Hub Protocol Adaptor Service are described as follows.

## Typical Internet of Things application scenarios
The customer has invested in the device with private data format and deployed a scale of stock devices. The customer hopes that the device data can perform JD Cloud Onboarding to fully utilize the data processing, storage, application and other services on the public cloud with the precondition of protecting the existing device investment.
