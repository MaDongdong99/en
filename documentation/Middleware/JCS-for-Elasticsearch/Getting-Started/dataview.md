## Data Management Overview
Open JD Cloud [JCS for Elasticsearch Console](https://es-console.jdcloud.com/clusters), click **Operation-kibana** to go to the kibana visual interface, enter Developer Tools through [Dev Tools] in the left navigation bar, and send all kinds of operation requests to clusters for data management operations.
