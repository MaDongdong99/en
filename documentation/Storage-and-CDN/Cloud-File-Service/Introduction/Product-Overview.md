# Product Overview

JD Cloud File Service(CFS) is a fully-hosted distributed file system with high reliability, expandability and sharable access. It can be expanded or reduced as needed based on your use of the file system without interrupting the application service, and billed according to actual usage. The NFS v4.1 and NFS v4.0 protocol is adopted, which supports POSIX file interface and provides shared access services for Linux Virtual Machines.
